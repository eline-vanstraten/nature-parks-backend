# 🌲 US National Parks API

Een RESTful webservice gebouwd met Express en MongoDB.

De API bevat data over natuurparken in de Verenigde Staten en maakt het mogelijk om deze op te vragen, toe te voegen, te
bewerken en te verwijderen.

# Technologieën

- Node.js
- Express.js
- MongoDB
- Mongoose

# Functies

- Alle natuurparken ophalen
- Eén park ophalen op ID
- Nieuw park toevoegen
- Park bewerken
- Park verwijderen

# Installatie

1. Clone de repository:

```sh
git clone https://github.com/eline-vanstraten/nature-parks-backend.git
```

2. Ga naar de map:

```sh
cd nature-parks-backend
```

3. Installeer dependencies:

```sh
npm install
```

4. Start de development server:

```sh
npm run dev
```
